# Spring Boot - Dependency Injection Demo

This project demonstrates the basics of Dependency Injection in Spring Boot.

## Concepts Covered

- What is Dependency Injection
- Inversion of Control (IoC)
- Spring Container
- Beans
- Component Scanning
- Constructor Injection

## Tech Stack

- Java
- Spring Boot
- Maven

## How to Run

Clone the repository and run:

```bash
mvn spring-boot:run
